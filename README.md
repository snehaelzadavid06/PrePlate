**PrePlate** is a modern, full-stack web application designed to revolutionize the college canteen experience. It streamlines food ordering, reduces wait times with smart queue management, and engages students through voting and AI assistance.

Built for **Google Developer Groups (GDG) Hackathon
Check it out at https://college-canteen-app.vercel.app/**.

## Features

###  For Students
- **Smart Menu & Ordering**: Browse a visual menu, filter by categories, and add items to a cart.
- **Time Slot Booking**: Select specific pickup slots to avoid overcrowding.
- **Real-Time Queue Status**: View live crowd levels (Low/Medium/High) and estimated wait times.
- **AI Chatbot Assistant**: Ask "PrePlate AI" (powered by **Google Gemini**) for food recommendations ("What's spicy?"), wait times, or today's specials.
- **Vote for Tomorrow**: Influence the next day's menu by voting on poll items.
- **QR Code Pickup**: Generate a unique QR code for every confirmed order for seamless verification at the counter.
- **Order History**: Track status (Pending, Served) of current and past orders.

###  For Admins
- **Live Order Dashboard**: View incoming orders in real-time with status indicators.
- **QR Scanner**: Built-in camera scanner to verify student orders instantly.
- **Menu Management**: Add, edit, or remove food items dynamically.
- **Poll Management**: Create new polls for future menu items and view voting results.
- **Analytics**: Track total revenue, total orders, and popular items.


##  Tech Stack

- **Frontend**: React.js (Vite), Tailwind CSS, Framer Motion (Animations).
- **Backend / Database**: Firebase Firestore (Real-time NoSQL DB).
- **AI Integration**: Google Gemini API (`gemini-1.5-flash`).
- **Icons**: Lucide React.
- **QR Tools**: `react-qr-code` (Generation), `html5-qrcode` (Scanning).
